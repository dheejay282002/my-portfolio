# Worship Presenter Pro

**Production-Grade Worship Presentation Software for Churches & Live Events**

A powerful, multi-display presentation system built with Electron, designed to manage and present lyrics, Bible verses, media, announcements, and countdown timers across multiple screens — all controllable from a single operator console or remotely via mobile/tablet.

---

## Features

### Multi-Display Output

| Window | Purpose |
| ------ | ------- |
| **Main Window** | Operator control interface — song library, Bible browser, media grid, slide editor, service plan builder |
| **Output Window** | Congregation-facing projection (fullscreen on external display) |
| **Stage Display** | Speaker-facing screen — current slide, next slide, clock, notes, chords, key |
| **Confidence Monitor** | Technical operator view — session timer, upcoming content, speaker notes |

### Presentation Engine
- Slide history with forward/back navigation
- Customizable transitions (fade, crossfade, etc.) and duration
- Blackout mode, logo screen mode
- Adjustable text formatting per slide
- Background support (solid colors, gradients, custom images)

### Song Management
- Full CRUD for songs with arrangements
- Multi-part support (verse, chorus, bridge, tag, intro, outro, interlude, pre-chorus)
- Chordpro-style chord lyrics
- Category filtering (Worship, Praise, Hymn, Contemporary, Gospel, Traditional)

### Bible Integration
- Multi-version support (KJV, NIV, NKJV, ESV, NLT)
- Book/chapter/verse navigation
- Full-text Bible search
- One-click "Present Verse" to output

### Media Library
- Import images, videos, audio files
- Grid-based media browser with thumbnail preview

### Remote Control
- Built-in Express + WebSocket server
- Control presentations from any device on the local network via browser
- Live state synchronization to all connected devices

---

## Tech Stack

| Layer | Technology |
| ----- | ---------- |
| **Runtime** | Electron |
| **Frontend** | Vanilla HTML/CSS/JS |
| **IPC Bridge** | contextBridge + ipcRenderer |
| **Database** | SQLite via sql.js (WAL mode, auto-migration) |
| **Remote Server** | Express + WebSocket (ws) |
