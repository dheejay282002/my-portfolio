A full-stack web application built with Next.js and PostgreSQL.

## Core Modules

| Module | File | Role |
| ------ | ---- | ---- |
| PresentationEngine | `core/presentation-engine.js` | Manages slide state, history, transitions |
| DisplayManager | `core/display-manager.js` | Detects and manages multiple displays |
| SongManager | `core/song-manager.js` | CRUD for songs, arrangements, chord lyrics |
| BibleEngine | `core/bible-engine.js` | Bible verse lookup, multi-version support |
| MediaEngine | `core/media-engine.js` | Imports and manages images, videos, audio |
| AudioEngine | `core/audio-engine.js` | Audio playback and volume control |
| ServicePlanner | `core/service-planner.js` | Builds ordered service plans |
| DatabaseManager | `database/db.js` | SQLite with auto-migration and seeding |

## Features

- Real-time collaboration
- Role-based access control
- Responsive design
- RESTful API
