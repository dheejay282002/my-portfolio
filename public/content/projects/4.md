# FLiX — Free Movie Streaming Platform

FLiX is a full-stack web application built with **Next.js 16**, **React 19**, **Tailwind CSS 4**, **Prisma**, and **SQLite**. It aggregates and streams free full-length movies sourced from YouTube, Vimeo, Dailymotion, and Archive.org.

## Features

- **Browse & Discover** — Movies organized by genre/category with a hero banner and infinite-style rows
- **Search** — Find movies by title or keyword
- **Watch** — HD video player with quality selector and related video suggestions
- **Continue Watching** — Tracks playback progress across sessions
- **My List** — Save movies to a personal watchlist
- **User Accounts** — Register, log in, and manage favorites (JWT + bcrypt)
- **Admin Panel** — Manage movies (CRUD) and API keys via a protected dashboard
- **Responsive UI** — Dark-themed Netflix-style design, adaptive to all screen sizes

## Tech Stack

| Layer | Technology |
| ----- | ---------- |
| Framework | Next.js 16 (App Router) |
| Frontend | React 19, Tailwind CSS 4, TypeScript |
| Backend | Next.js API Routes + middleware |
| Database | SQLite via Prisma ORM |
| Auth | JWT (jsonwebtoken) + bcryptjs |
| Video APIs | YouTube, Vimeo, Dailymotion, Archive.org |
